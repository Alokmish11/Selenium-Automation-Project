# Selenium Automation Project - TechBridgeAssignment

This project is a Selenium Automation Testing framework built using Java, TestNG, and Maven. It follows the Page Object Model (POM) design pattern for better maintainability and scalability.

---

## 🚀 Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Eclipse IDE
- Git & GitHub

---

## 📂 Project Structure

src/
├── main/java
├── test/java
│   ├── Base
│   ├── Pages
│   └── Tests
│
├── test/resources
│
pom.xml  
testng.xml  

---

## ✅ Features Automated

- User Registration Test
- Form Validation Testing
- Positive & Negative Scenarios
- TestNG Execution
- Basic Framework Setup
